# 更新日志

## v2.8.5（2026-07-07）

### 🆕 新增功能

1. **工具栏自定义显示** — 设置页新增「工具栏」标签页，16 项功能可独立勾选显示/隐藏，支持全选/全不选，设置持久化保存
2. **工具栏与规范库碰撞自动避让** — 窗口收窄时规范库自动下移不遮挡工具栏，拉宽后自动归位
3. **边缘拖动缩放** — 规范库右侧和工具栏右侧均支持拖拽改变宽度（规范库 220–450px，工具栏 320–视口宽−48）
4. **规范库二级分类折叠动效** — 餐饮工装/酒店民宿/家装展开/收起增加平滑过渡动画

### 🐛 Bug 修复

- **工具栏点击失效** — 移除旧版 `pointer-events: none` 容器样式，所有按钮恢复可点击
- **死代码清理** — 移除 `KNOWN_VERSIONS` 未使用数组（Marvis 审查建议）

### 🎨 UI/UX

- 规范库顶部高度再降低，底部圆角正确裁剪
- 画布全屏铺满，工具栏/规范库/AI 面板改为悬浮磁贴（消除边界感）
- ZoomControls 从画布右上角移到右下角 AI 气泡旁

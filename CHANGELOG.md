# 更新日志

## v2.9.1（2026-07-09）— 规范库外置化（P0）

> 规范数据不再写死在前端，改由「内置兜底 + 远程热更新」驱动。**改规范不再需要重新打包安装，用户无感升级**。无需自有服务器——直接复用 GitHub 仓库作为规范源。

### 🆕 新增功能

1. **规范库外置化** — 45 品类 / 1087 条规范迁至独立 `spec.json`（带 `version` 字段），EXE 内仅保留一份内置兜底
2. **启动热更新** — 启动时按「内置兜底 → userData 缓存 → 远程源」顺序加载；远程版本更新则自动覆盖并写入本地缓存
3. **双源容错** — 远程源默认 jsDelivr 镜像，GitHub raw 自动兜底；网络不可达时静默回退内置/缓存，规范库永不空
4. **规范源可配置** — 设置页新增「规范」标签页，可自定义规范源地址（支持任意可访问的 spec.json），并一键「立即检查更新」
5. **版本可见** — 规范库面板标题栏显示当前生效规范版本与来源

### 🔧 技术实现

- `src/data/specs.js` 重构为运行时数据层（保留全部 getter 签名，消费方零改动）
- `src/data/specs.bundled.js` 内置兜底数据（45 品类 / 1087 条规范）
- `public/spec.json` 远程源基线（提交仓库后通过 jsDelivr / GitHub raw 分发）
- Electron `userData/spec-cache.json` 离线缓存
- `scripts/gen-spec.mjs` 从内置数据导出远程 `spec.json`

## v2.9.0（2026-07-09）— 画板体验对齐 PureRef

> 依据 Marvis 的「ID Aura 图板 vs PureRef」对比报告补齐高频直觉操作流。

### 🆕 新增功能（画板体验深化）

1. **空格 + 拖动平移** — 按住空格画布进入抓取模式（光标 `grab`/`grabbing`），单手即可平移，对齐 Figma/PS/PureRef 肌肉记忆；保留中键 / Alt 拖动兼容
2. **方向键微移** — 选中卡片后方向键 1px 精调，`Shift+方向键` 10px；连续按住只产生一个撤销步
3. **F 键适配视图** — `F` 一键适配全部内容，`Shift+F` 重置 100%
4. **单图透明度** — 选中单张卡片时底部浮现属性条，滑块调节不透明度（0–100%），用于叠加对比参考
5. **组折叠** — 右键编组 →「折叠组」可将整组收为悬浮小图标（📁 组·N 项），双击或点「展开」还原；折叠态下拖拽小图标即可整体移动整组
6. **最小化到参考小窗** — 右键画布 →「最小化到参考小窗」，开一个常驻置顶的简化参考窗（实时快照同步），可一键还原主窗口（PureRef 标志性工作流）
